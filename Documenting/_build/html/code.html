class card.Card(suit, number)¶

The card class represents a single playing card and is initialised by passing a suit and number.

number¶

Gets the number of the card in the form 2-10 or J Q K A

suit¶

Gets or sets the suit of the card

class card.Deck¶

The Deck class represents a deck of playing cards in order

deal(no_of_cards)¶

Deals (and removes) a number of cards from the deck

populate()¶

Populate (resets) the deck of cards with all 52 cards in the order 2-10, J, Q, K, A, Hearts, Clubs, Diamonds, Spades

shuffle()¶

Shuffles the deck in a random order of playing cards
